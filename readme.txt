Исполняемый файл - sitesoft-testcase-0.0.1-jar-with-dependencies.jar
Собран под java 1.8

При первом запуске нужно подать команду initialization.
При этом в текущем каталоге создастся база данных в файле с именем Documents. Повторная команда initialization пересоздаст базу данных.

Работа приложения завершается, когда пользователь подаст команду quit.

Приложение может выполнять следующие команды:
Выдать список документов, отсортированных по индексу: documents
Показать документ: document <id документа>
Добавить документ: add document <индекс документа> <название документа> <тип документа> <имя файла документа> 
Удалить документ: remove document <id документа>
Получить содержание документа: content <id документа> <имя файла, куда выгрузить контент>
Выдать атрибуты документа: attributes <id документа>
Добавить атрибут документу: add attribute <id документа> <название атрибута> <тип атрибута> <строковая величина> <целочисленная величина(может отсутствовать)>
Изменить атрибут документа: change attribute <id атрибута> <название атрибута> <тип атрибута> <строковая величина> <целочисленная величина(может отсутствовать)>
Удалить атрибут документа: remove attribute <id атрибута>
